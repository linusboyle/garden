# 谁在为中国老年患者“供血”？美国低收入人群的隐秘生意  端傳媒 Initium Media


Created: [[2025-06-27]] 

> 中国本土的采浆能力却难以跟上。人血浆是生产白蛋白的核心原料，而中国对血浆采集的监管极为严格。与采集血液全部成分的全血捐献不同，血浆采集会通过分离技术仅提取血浆，红细胞和血小板等成分会回输至献血者体内。受1990年代河南“血浆经济”丑闻的影响，过去24年来，中国政府未批准新的血液制品生产企业。中国内地的血液制品生产企业不足30家，新设血浆站的难度也比较高。
