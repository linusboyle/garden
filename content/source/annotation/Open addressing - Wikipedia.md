# Open addressing - Wikipedia


Created: [[2025-05-03]] 

> Open addressing, or closed hashing, is a method of collision resolution in hash tables.  With this method a hash collision is resolved by probing, or searching through alternative locations in the array (the probe sequence) until either the target record is found, or an unused array slot is found, which indicates that there is no such key in the table
