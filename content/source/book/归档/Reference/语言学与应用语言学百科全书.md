# 语言学与应用语言学百科全书
《语言学与应用语言学百科全书》系上海外国语大学重大科研项目暨教育部"211工程三期验收"重点项目，为国内外语界学者编写的规模较大、学科覆盖面较全、内容较新的一部语言学与应用语言学有机结合的综合性、百科性词典；达成了两个与众不同的目标，一是将语言学知识与应用语言学知识集于一身，二是将国外语言学学科知识与我国语言学学科知识融为一体，因而具有其独特的参考和实用价值。, 《语言学与应用语言学百科全书》以词典学、专科辞书编纂学、术语学以及语言学等理论为指导，借鉴国内外语言学专科辞书编纂的成功经验，致力于编纂一部内容覆盖面广、释义准确清晰、相关百科知识丰富、宏观结构合理、体例规范统一、参照结构完善严密、专业性、权\*性强的中型综合性语言学百科全书。, 词典主体部分由语言学术语、语言学理论与流派、语言学人物和世界主要语言四个板块构成。内容上中西兼顾，古今兼收。注重我国语..., (展开全部)
