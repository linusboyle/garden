# Vibe Coding

2025 年 2 月 Andrej Karpathy提出来的时髦词汇，其实就是用 AI（尤其是 [[大语言模型|LLM]]）通过自然语言进行项目构建。

关于Vibe Coding 的一种定义认为，应该完全避免查看生成的代码，用户只需要不断地测试-精化。 

> If an LLM wrote every line of your code, but you've reviewed, tested, and understood it all, that's not vibe coding in my book—that's using an LLM as a typing assistant.
---

虽然在早期Vibe Coding受到了程序员的质疑，但是在当下（[[2026-03-12]]），编程智能体的能力已经得到了印证，人们已经很快接受了这种新的编程方式。
