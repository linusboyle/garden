# Cartesian closed category

满足以下性质的范畴：

1.  有terminal object
2.  任意两个object的[[product|product]]和[[exponential object|exponential object]]都存在

如果再满足：

1.  有initial object
2.  任意两个object的coproduct object都存在

称为bi-Cartesian closed category
