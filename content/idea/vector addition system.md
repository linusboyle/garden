# vector addition system
A _VAS_ is a finite set $V \subseteq \mathbb{Z}^d$ for some $d\geq 1$

Let $V\subseteq \mathbb {Z} ^{d}$ be a VAS. Given a vector ${\displaystyle u\in \mathbb {N} ^{d}}$, the vector ${\displaystyle u+v}$ can be _reached_, in one transition, if ${\displaystyle v\in V}$ and ${\displaystyle u+v\in \mathbb {N} ^{d}}$
