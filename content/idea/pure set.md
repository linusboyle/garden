A pure set is a set whose elements are all pure. The inductive definition is only well-defined when $\in$ is well-founded.

 > all elements of the set are themselves sets, as are all elements of the elements, and so on.
 
 The empty set is [[202403301926 vacuous truth|vacuously]] a pure set.
