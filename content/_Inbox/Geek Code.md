From Wikipedia, the free encyclopedia

[![](https://upload.wikimedia.org/wikipedia/commons/thumb/c/c5/C%C3%B3digo_geek%2C_AA_2012.svg/290px-C%C3%B3digo_geek%2C_AA_2012.svg.png)](https://en.wikipedia.org/wiki/File:C%C3%B3digo_geek,_AA_2012.svg)

Geek code example, which opens by stating that the user is a Geek of Technical Writing (GTW) who usually wears jeans and a t-shirt (d-@), is of average height and above-average weight (s:+), and is aged between 25 and 29 (a-).

The **Geek Code**, developed in 1993, is a series of letters and symbols used by self-described "[geeks](https://en.wikipedia.org/wiki/Geek "Geek")" to inform fellow geeks about their [personality](https://en.wikipedia.org/wiki/Personality_psychology "Personality psychology"), appearance, interests, skills, and opinions. The idea is that everything that makes a geek [individual](https://en.wikipedia.org/wiki/Individual "Individual") can be [encoded](https://en.wikipedia.org/wiki/Code "Code") in a compact format which only other geeks can read. This is deemed to be [efficient](https://en.wikipedia.org/wiki/Algorithmic_efficiency "Algorithmic efficiency") in some sufficiently geeky manner.<sup id="cite_ref-1"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-1"><span>[</span>1<span>]</span></a></sup>

It was once common practice to use a geek code as one's email or Usenet signature, but the last official version of the code was produced in 1996, and it has now largely fallen out of use.<sup id="cite_ref-wapo_2-0"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-wapo-2"><span>[</span>2<span>]</span></a></sup><sup id="cite_ref-geekcode_3-0"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-geekcode-3"><span>[</span>3<span>]</span></a></sup>

A number of similar codes were developed for other subcultures, such as a Goth Code for the [Goth subculture](https://en.wikipedia.org/wiki/Goth_subculture "Goth subculture"), and the [Natural Bears Classification System](https://en.wikipedia.org/wiki/Natural_Bears_Classification_System "Natural Bears Classification System") for gay men.

The Geek Code was invented by Robert A. Hayden in 1993 and was defined at geekcode.com.<sup id="cite_ref-geekcode_3-1"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-geekcode-3"><span>[</span>3<span>]</span></a></sup> It was inspired by [a similar code](https://en.wikipedia.org/wiki/Natural_Bears_Classification_System "Natural Bears Classification System") for the [bear](https://en.wikipedia.org/wiki/Bear_(gay_culture) "Bear (gay culture)") subculture - which in turn was inspired by the [Yerkes spectral classification](https://en.wikipedia.org/wiki/Yerkes_spectral_classification_scheme "Yerkes spectral classification scheme") system for describing stars.<sup id="cite_ref-jargon_4-0"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-jargon-4"><span>[</span>4<span>]</span></a></sup><sup id="cite_ref-5"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-5"><span>[</span>5<span>]</span></a></sup><sup id="cite_ref-6"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-6"><span>[</span>6<span>]</span></a></sup>

After a number of updates, the last revision of the code was v3.12, in 1996.<sup id="cite_ref-7"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-7"><span>[</span>7<span>]</span></a></sup>

Some alternative encodings have also been proposed. For example, the 1997 Acorn Code was a version specific to users of [Acorn](https://en.wikipedia.org/wiki/Acorn_Computers "Acorn Computers")'s [RISC OS](https://en.wikipedia.org/wiki/RISC_OS "RISC OS") computers.<sup id="cite_ref-8"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-8"><span>[</span>8<span>]</span></a></sup>

Geek codes can be written in two formats;<sup id="cite_ref-geekcode_3-2"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-geekcode-3"><span>[</span>3<span>]</span></a></sup> either as a simple string:  

```
<span></span><span>GED</span><span>/</span><span>J</span><span> </span><span>d</span><span>--</span><span> </span><span>s</span><span>:</span><span>++&gt;</span><span>:</span><span> </span><span>a</span><span>--</span><span> </span><span>C</span><span>++</span><span>(</span><span>++++</span><span>)</span><span> </span><span>ULU</span><span>++</span><span> </span><span>P</span><span>+</span><span> </span><span>L</span><span>++</span><span> </span><span>E</span><span>----</span><span> </span><span>W</span><span>+</span><span>(</span><span>-</span><span>)</span><span> </span><span>N</span><span>+++</span><span> </span><span>o</span><span>+</span><span> </span><span>K</span><span>+++</span><span> </span><span>w</span><span>---</span><span> </span><span>O</span><span>-</span><span> </span><span>M</span><span>+</span><span> </span><span>V</span><span>--</span><span> </span><span>PS</span><span>++&gt;$</span><span> </span><span>PE</span><span>++&gt;$</span><span> </span><span>Y</span><span>++</span><span> </span><span>PGP</span><span>++</span><span> </span><span>t</span><span>-</span><span> </span><span>5</span><span>+++</span><span> </span><span>X</span><span>++</span><span> </span><span>R</span><span>+++&gt;$</span><span> </span><span>tv</span><span>+</span><span> </span><span>b</span><span>+</span><span> </span><span>DI</span><span>+++</span><span> </span><span>D</span><span>+++</span><span> </span><span>G</span><span>+++++</span><span> </span><span>e</span><span>++</span><span> </span><span>h</span><span> </span><span>r</span><span>--</span><span> </span><span>y</span><span>++**</span>
```

...or as a "Geek Code Block", a parody of the output produced by the encryption program [PGP](https://en.wikipedia.org/wiki/Pretty_Good_Privacy "Pretty Good Privacy"):  

```
<span></span><span>-----BEGIN GEEK CODE BLOCK-----</span>
<span>  </span><span>Version:</span><span> </span><span>3.1</span>
<span>  </span><span>GED/J</span><span> </span><span>d--</span><span> </span><span>s:++&gt;:</span><span> </span><span>a--</span><span> </span><span>C++(++++)</span><span> </span><span>ULU++</span><span> </span><span>P+</span><span> </span><span>L++</span><span> </span><span>E----</span><span> </span><span>W+(-)</span><span> </span><span>N+++</span><span> </span><span>o+</span><span> </span><span>K+++</span><span> </span><span>w---</span>
<span>  </span><span>O-</span><span> </span><span>M+</span><span> </span><span>V--</span><span> </span><span>PS++&gt;$</span><span> </span><span>PE++&gt;$</span><span> </span><span>Y++</span><span> </span><span>PGP++</span><span> </span><span>t-</span><span> </span><span>5+++</span><span> </span><span>X++</span><span> </span><span>R+++&gt;$</span><span> </span><span>tv+</span><span> </span><span>b+</span><span> </span><span>DI+++</span><span> </span><span>D+++</span>
<span>  </span><span>G+++++</span><span> </span><span>e++</span><span> </span><span>h</span><span> </span><span>r--</span><span> </span><span>y++**</span>
<span>-----END GEEK CODE BLOCK-----</span>
```

Note that this latter format has a line specifying the version of Geek Code being used.

(Both these examples use Hayden's own geek code.)<sup id="cite_ref-jargon_4-1"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-jargon-4"><span>[</span>4<span>]</span></a></sup>

The code starts with the letter `G` (for Geek) followed by the geek's occupation(s): `GMU` for a geek of [music](https://en.wikipedia.org/wiki/Music "Music"), `GCS` for a geek of computer science etc. There are 28 occupations that can be represented, but `GAT` is for geeks that can do anything and everything - and "usually precludes the use of other vocational descriptors".<sup id="cite_ref-geekcode_3-3"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-geekcode-3"><span>[</span>3<span>]</span></a></sup>

The Geek Code website contains the complete list of categories, along with all of the special syntax options.<sup id="cite_ref-9"><a href="https://en.wikipedia.org/wiki/Geek_Code#cite_note-9"><span>[</span>9<span>]</span></a></sup>
